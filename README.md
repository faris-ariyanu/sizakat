# Sistem Informasi Zakat

> Laravel + CoreUI + Vue Boilerplate

## How to Install
1. Clone this project
```bash
git clone https://github.com/faris-ariyanu/sizakat.git
```
2. Install Dependencies
```bash
composer install
npm install
```
3. Import Database
```bash
php artisan migrate
```
4.  Compile Static Asset
```bash
## for Development
npm run dev

## for Production
npm run prod

## for Development with HMR (Hot Module Replacement)
npm run hot
```
